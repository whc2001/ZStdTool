# ZStdTool
 Zstd compress and decompress from context menu

# Usage
 Install to context menu: `ZStdTool.exe /s`

 Uninstall from context menu: `ZStdTool.exe /r`

 Compress: `ZStdTool.exe /c file`

 Decompress: `ZStdTool.exe /d file`
